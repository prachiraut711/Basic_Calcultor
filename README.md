# 🧮 Basic Calculator – Web Application

Basic Calculator is a simple and interactive web-based application that performs arithmetic operations like addition, subtraction, multiplication, and division. It is designed to demonstrate fundamental JavaScript concepts and user interface interaction.

---

## 🚀 Features

- ➕ Addition  
- ➖ Subtraction  
- ✖️ Multiplication  
- ➗ Division  
- 🧹 Clear/Reset functionality  
- ⚡ Real-time input and result display  
- 🖥️ Clean and responsive user interface  

---

## 🛠️ Tech Stack

### 🌐 Frontend
- HTML  
- CSS  
- JavaScript  

---

## 🧠 How It Works

1. User inputs numbers using calculator buttons  
2. Selected operation is stored  
3. JavaScript processes the calculation  
4. Result is displayed instantly on screen
